Unit Converter guidance

Perform fast conversions across engineering units with category-aware validation.

Using the converter

  1. Select a measurement category (length, mass, temperature, etc.).
  2. Enter a numeric value. Scientific notation is accepted.
  3. Choose source and destination units; incompatible combinations are filtered automatically.
  4. Submit the form to view the converted value and precision-controlled output.

Accuracy notes

  • Conversions use vetted factors stored in plugins/unit_converter/core; update these tables to extend coverage.
  • Results are rounded to four decimal places to keep laboratory notebooks tidy.
  • Use the reset button to clear the workspace before switching categories.
